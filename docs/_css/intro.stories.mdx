# Vanilla HTML + CSS

Castor's lowest level API.

Currently the only option in case you are not using React, or need more
flexibility than what its components offer.

## Usage

Components are CSS classes:

```html dark
<button class="ods-button -action--primary">Hello world!</button>
```

Helper functions are available from the main package:

```js dark
import { space } from '@onfido/castor';

document.body.style.padding = space(2);
```
