# React

Currently our most complete and easy to use API.

## Icons

If you plan to use icons, we recommend inlining an svg sprite closest to your
app's root:

```tsx dark
import { Icons } from '@onfido/castor-icons';

export const App = () => (
  <>
    <Icons />
    {/* the rest of your app here */}
  </>
);
```

It contains all symbols and will make using icons much simpler via the `Icon`
component:

```tsx dark
import { Icon } from '@onfido/castor-react';

<Icon name="alarm" />;
```
